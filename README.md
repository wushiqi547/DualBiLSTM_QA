# DualBiLSTM_QA
Q&amp;A matching model use DualBiLSTM


# data
数据为公司提供的游戏类问答对数据
格式：玩家问题 ---- 标准问题 `6000+条问答对`


# training
用训练数据进行相似度训练，用验证集数据验证保存最优模型
